All Themes

This is a theme directory for the NetNewsWire RSS reader. NetNewsWire 6.1 or newer required for themes.

  • Broadsheet

    • Stuart Breckenridge
    • User Theme
    • Light and Dark

    If NetNewsWire were a newspaper—and it if were, it’d be a quality read—it might look like this. The New York typeface is used throughout and the favicon has been removed.

  • Druckpresse

    • Hans Schülein
    • User Theme
    • Light and Dark

    A modification of Appanoose with the New York system font to achieve a design inspired by classic print newspapers.

  • Default

    • Maurice Parker
    • Default Theme
    • Light and Dark

    This theme is not actually shipped with NetNewsWire as a theme and has instead been reconstructed from the HTML and CSS in the app bundle. This is the theme that is preset in NetNewsWire.

  • Newsfax

    • Stuart Breckenridge
    • Default Theme
    • Dark Only

    A modified version of the Promenade theme, inspired by Teletext and Ceefax services of yesteryear. It comes complete with Teletext style page numbers and R/G/B/Y options along the bottom of each article. Remote control not included.

  • Promenade

    • Stuart Breckenridge
    • Default Theme
    • Light and Dark

    Prominent titles, snazzy blockquotes, and simplified external links. Charter and Avenir for your viewing pleasure.

  • Sepia
